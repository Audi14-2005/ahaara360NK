Your CSV Format

Your CSV file should have these columns (exactly as shown in your file):

Your Column Required Description Example
id Optional Food ID (will be ignored) 1, 2, 3...
Food Item Required Food name Basmati Rice, Apple, etc.
Meal Type Optional Meal type Breakfast, Lunch, Dinner, Snack
Calories (k) Required Calories per 100g 130, 52, etc.
Protein (g) Required Protein in grams 2.7, 0.3, etc.
Carbs (g) Required Carbohydrates in grams 28.0, 13.8, etc.
Fat (g) Required Fat in grams 0.3, 0.2, etc.
Fibre (g) Required Fiber in grams 2.4, 1.8, etc.
Category Optional Food category Grains, Fruits, Vegetables, etc.
Tags Optional Food tags healthy, organic, spicy, etc.
Rasa Required Primary taste (Ayurvedic) Sweet, Sour, Salty, Pungent, Bitter, Astringent
Virya Required Energy property Heating, Cooling, Neutral
Vipaka Optional Post-digestive effect Sweet, Sour, Pungent
Guna Optional Qualities Light, Heavy, Dry, Oily, Cold, Hot
Vata Effect Required Effect on Vata dosha Pacifies, Aggravates, Neutral
Pitta Effect Required Effect on Pitta dosha Pacifies, Aggravates, Neutral
Kapha Effect Required Effect on Kapha dosha Pacifies, Aggravates, Neutral
